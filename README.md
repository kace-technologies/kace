## 🚀 Getting Started

### Prerequisites

- **Node.js** (v18 or higher)
- **Bun** (optional, for faster development)
- **Git**

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/kace-technologies/kace.git
   cd kace
   ```

2. **Install dependencies**
   ```bash
   # Using npm
   npm install

   # Or using Bun (recommended for faster installs)
   bun install
   ```

3. **Start the development server**
   ```bash
   # Using npm
   npm run dev

   # Or using Bun
   bun run dev
   ```

4. **Open your browser**
   
   Navigate to `http://localhost:5173` to view the application.
